# ros-code-checker
ROS Code Checker & Simulation Preview

This repo implements a code checker, a simulation runner (Gazebo), and a minimal Flask web UI for uploading and checking ROS packages.

# Quick start (development host with ROS/Gazebo installed)

1. Install dependencies for checker and webapp:


```bash
python3 -m pip install -r checker/requirements.txt
python3 -m pip install -r webapp/requirements.txt
